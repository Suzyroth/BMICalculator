BMI Calculator (Python)

Project Description
This project is a BMI (Body Mass Index) Calculator developed in Python as part of my internship. It includes both a basic command-line version and an advanced GUI-based application.

The application calculates BMI using user input (height and weight) and classifies the result into standard health categories. The advanced version provides a graphical interface, data storage, and visualization features.

Features

Basic Version (CLI)

Takes user input (weight in kg, height in meters)
Calculates BMI
Displays BMI result
Categorizes BMI as Underweight, Normal weight, Overweight, or Obese

Advanced Version (GUI)

User-friendly graphical interface
Input fields with validation
Instant BMI calculation
Displays BMI category clearly
Stores user data
Shows historical BMI records
Visualizes BMI trends using graphs

BMI Formula
BMI = weight (kg) / (height (m))^2

BMI Categories
Less than 18.5 – Underweight
18.5 to 24.9 – Normal weight
25 to 29.9 – Overweight
30 and above – Obese
